  • JLox Programming Language

    Java, Elm, Spring Boot, graphQL, HTML, CSS

    Wrote code for a scanner, parser, interpreter, resolver, and other required classes to implement the Lox programming language that is described in the "Crafting Interpreters" book. This backend implementation was connected to a frontend web page that was written in Elm where code could be entered and executed using Lox language syntax. I also utilized Spring Boot and graphQL to manage dependencies, XML configuration, and to generate classes.
